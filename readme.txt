INTRODUCTION
------------

This plugin is meant to spare users from having to manually insert the Readefined.com Javascript snippet in their Wordpress HTML. The plugin will automatically generate the code and embed it correctly inside the HEAD tag.


INSTALLATION PREREQUISITES
--------------------------

The plugin requires that you have an active Readefined.com account and that you have correctly configured your Readefined.com property to include the domain name that matches your Wordpress site domain. If your site is "domain.com", then the DOMAIN field of your Readefined.com property must be "domain.com".The plugin will send the domain of the web site it is deployed on to Readefined.com to retrieve the Javascript embed code to include in your site's HEAD tag.
